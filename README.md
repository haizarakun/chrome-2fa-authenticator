# クローム2段階認証（Chrome 2FA Authenticator）

Google Authenticatorの移行用データ（`otpauth-migration://`）やQRコードからアカウントを簡単にインポートし、ブラウザ上で安全に2FA（二段階認証）コードを生成・管理・エクスポートできるGoogle Chrome拡張機能です。

## 特徴

- **完全ローカル動作 (100% オフライン)**: データの保存にはChromeのローカルストレージ（`chrome.storage`）のみを使用します。インポート・QRコード生成・カメラ読み取りを含め、外部サーバーへの通信やデータ送信は一切行いません。
- **マスターパスワードによる暗号化保存**: アカウントの秘密鍵（シークレット）は平文では保存されません。マスターパスワードからPBKDF2(20万回)で導出した鍵でAES-GCM暗号化した状態でのみディスクに保存されます。復号後のデータはポップアップを開いている間だけメモリ上に存在します。
  - **マスターパスワードを忘れると復元できません**（バックドアなし・意図的な設計です）。
  - 一度ロック解除すると、同じブラウザセッション中（ブラウザを閉じるまで）は再入力を求められません。手動でロックしたい場合はタイトル行の🔒ボタンを押してください。
- **2通りのインポート方法**
  - **カメラでQR読み取り**: PC/USBカメラで移行用QRコードや通常のotpauthQRコードを直接スキャンできます（複数カメラがある場合は選択可能）。完全ローカルで解析（`jsQR`使用、外部送信なし）。
  - **テキストから読み込む**: `otpauth-migration://...` または `otpauth://totp/...` のURI文字列を直接貼り付けてインポートできます。カメラが無い環境やバックアップ復元時に利用してください。
- **1クリックでコードをコピー**: 認証コードをクリックするとクリップボードにコピーされます。他の拡張機能に読み取られるリスクを抑えるため、約30秒後に自動でクリップボードを消去します（消去前に別の内容へ上書きされていた場合は消去しません）。
- **安全なQRコードエクスポート**: 登録したアカウントを他のデバイスに移行するためのQRコードを、外部APIを使わずPC内（`qrcode.min.js`）で完全ローカル生成します。
- **テーマカラー変更**: アクセントカラーと背景カラーをそれぞれカラーピッカーで自由に変更できます。
- **操作ログ**: インポート・削除・コピーなどの操作履歴を確認・コピー・消去できます（シークレットや認証コード自体は記録されません）。
- **アカウント管理**: 不要になったアカウントの個別削除に対応しています。
- **Manifest V3 準拠**: 最新の Google Chrome 拡張機能規格に合わせて最適化されています。

## インストール方法

1. このリポジトリの右上にある「**Code**」ボタンから「**Download ZIP**」をクリックしてダウンロードし、任意の場所に解凍（展開）します。
2. Google Chrome を開き、アドレスバーに `chrome://extensions/` と入力して拡張機能管理画面を開きます。
3. 画面右上の「**デベロッパーモード**」をオンにします。
4. 画面左上の「**パッケージ化されていない拡張機能を読み込む**」をクリックします。
5. 解凍したフォルダ内にある、`manifest.json` が直下に入っている方の「**2FA**」フォルダを選択するとインストールが完了します。

## 使い方

### 初回起動（マスターパスワードの設定）
1. インストール後、初めて拡張機能アイコンを開くと「マスターパスワードを設定」画面が表示されます。
2. パスワードを2回入力して設定します。**このパスワードはどこにも保存されません。忘れると保管データを復元できないため、ご自身での管理をお願いします。**
3. 旧バージョンから引き続き利用している場合、既存のアカウントは自動的に暗号化保存へ移行されます。

### アカウントのインポート
- **カメラで読み取る場合**: 「カメラでQRを読み取る」ボタンを押すと別ウィンドウが開きます。カメラを選択し、Google Authenticatorなどの「アカウントの移行」QRコード、または通常の`otpauth://totp/...` QRコードをカメラに映してください。自動で検出・インポートされます。
- **テキストで読み込む場合**: 移行用URI（`otpauth-migration://...`）または単体のURI（`otpauth://totp/...`）を下部の入力欄に貼り付けて「**テキストから読み込む**」ボタンを押します。

### コードの確認と管理
- **コードの確認・コピー**: 登録されたアカウントの2FAコードが30秒ごとに自動生成されます。コード部分をクリックするとクリップボードにコピーされます（約30秒後に自動消去）。
- **アカウントの削除**: 枠内の「**削除**」ボタンから個別に削除できます。
- **ロック**: タイトル行の🔒ボタンで即座にロックできます。次回はマスターパスワードの再入力が必要です。

### 他デバイスへの再移行（エクスポート）
- 各アカウントの枠内にある「**QR表示**」ボタンを押すと、その場に移行用のQRコードが完全オフラインで生成されます。スマートフォンなどの認証アプリでこのQRコードをスキャンすれば、簡単にアカウントを移行できます。

### 見た目のカスタマイズ
- タイトル行のカラーピッカー2つで、アクセントカラーと背景カラーをそれぞれ変更できます。

## セキュリティに関する既知の制限

- OSレベルでのメモリダンプなど、ブラウザプロセス自体を直接解析できる攻撃者には対抗できません（ブラウザ拡張機能の権限モデル上の限界です）。
- HOTP（イベント同期方式）には対応していません。TOTP（時間同期方式）のみ対応です。
- クリップボード自動消去のタイミング（約30秒）はブラウザの`alarms` APIの仕様上、多少前後する場合があります。
