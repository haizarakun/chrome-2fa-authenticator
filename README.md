# クローム2段階認証 (Chrome 2FA Authenticator)

Google Authenticatorの移行用データ（`otpauth-migration://`）からアカウントを簡単にインポートし、ブラウザ上で安全に2FA（二段階認証）コードを生成・管理できるGoogle Chrome拡張機能です。

## 特徴

- **Chrome内で完全完結:** ブラウザのポップアップから、いつでも1クリックで認証コード（TOTP）を確認・コピーできます。
- **完全ローカル動作 (オフライン):** データの保存にはChromeのローカルストレージ（`chrome.storage`）のみを使用し、外部サーバーへの通信やデータ送信は一切行いません。秘密鍵がネット上に流出するリスクがなく、安全です。
- **安全な設計:** インポートしたデータや画面への描画には、XSS（クロスサイトスクリプティング）などの脆弱性を排除した安全な処理を採用しています。
- **Manifest V3 準拠:** 最新のGoogle Chrome拡張機能規格に合わせて最適化されています。

## インストール方法

1. このリポジトリの右上の「**Code**」ボタンから「**Download ZIP**」をクリックしてダウンロードし、解凍します。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` を入力して拡張機能管理画面を開きます。
3. 画面右上の「**デベロッパー モード**」をオンにします。
4. 画面左上の「**パッケージ化されていない拡張機能を読み込む**」をクリックします。
5. 解凍した `2FA` フォルダー（`manifest.json` が入っているフォルダー）を選択すると、インストールが完了します。

## 使い方

1. Google Authenticatorなどのアプリから移行用QRコードを出力し、そのURI（`otpauth-migration://...`）の文字列を取得します。
2. 本拡張機能のポップアップを開き、取得したURIを貼り付けてインポートします。
3. 登録されたアカウントの2FAコードが30秒ごとに自動生成されます。
