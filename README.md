# クローム2段階認証（Chrome 2FA Authenticator）

Google Authenticatorの移行用データ（`otpauth-migration://`）からアカウントを簡単にインポートし、ブラウザ上で安全に2FA（二段階認証）コードを生成・管理・エクスポートできるGoogle Chrome拡張機能です。

## 特徴

- **1クリックでコードを確認**: ブラウザのポップアップから、いつでも即座に認証コード（TOTP）を確認・コピーできます。
- **完全ローカル動作 (100% オフライン)**: データの保存にはChromeのローカルストレージ（`chrome.storage`）のみを使用します。インポートやQRコード生成を含め、外部サーバーへの通信やデータ送信は一切行わないため、秘密鍵がネット上に流出するリスクがありません。
- **安全なQRコードエクスポート**: 登録したアカウントを他のデバイスに移行するためのQRコードを、外部APIを使わずPC内（`qrcode.min.js`）で安全に完全ローカル生成します。
- **アカウント管理**: 不要になったアカウントの個別削除に対応しています。
- **Manifest V3 準拠**: 最新の Google Chrome 拡張機能規格に合わせて最適化されています。

## インストール方法

1. このリポジトリの右上にある「**Code**」ボタンから「**Download ZIP**」をクリックしてダウンロードし、任意の場所に解凍（展開）します。
2. Google Chrome を開き、アドレスバーに `chrome://extensions/` と入力して拡張機能管理画面を開きます。
3. 画面右上の「**デベロッパーモード**」をオンにします。
4. 画面左上の「**パッケージ化されていない拡張機能を読み込む**」をクリックします。
5. 解凍したフォルダ内にある、`manifest.json` が直下に入っている方の「**2FA**」フォルダを選択するとインストールが完了します。

## 使い方

### アカウントのインポート（移行）
1. Google Authenticator などのアプリから「アカウントの移行」を選び、移行用QRコードを出力します。
2. そのQRコードをスキャン等で読み取り、`otpauth-migration://...` から始まるURIの文字列を取得します。
3. 本拡張機能のポップアップを開き、下部の入力欄に取得したURIを貼り付けて「**データを同期**」ボタンを押します。

### コードの確認と管理
- **コードの確認**: 登録されたアカウントの2FAコードが30秒ごとに自動生成されます。
- **アカウントの削除**: 不要になったアカウントは、枠内の「**削除**」ボタンから個別に削除できます。

### 他デバイスへの再移行（エクスポート）
- 各アカウントの枠内にある「**QR表示**」ボタンを押すと、その場に移行用のQRコードが完全オフラインで生成されます。スマートフォンなどの認証アプリでこのQRコードをスキャンすれば、簡単にアカウントを移行できます。
